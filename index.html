Whats A Font?

  • A "Typeface" is a collection of fonts with different weights like bold and styles like italic

  • A Typeface may also be known as a font, just to confuse things

  • Each font is made up of graphically related glyphs, characters, or symbols

  • Each glyph is identified by a unicode mapping (143,859 as of Unicode 13.0)

Categories

  • SerifParagraphs of text, great for readability
  • Sans SerifGood for titles, headlines as well as body copy
  • DisplayTitles again, have a bit of fun
  • HandwritingStyle your bad poetry

Code: System Fallbacks

System Fonts with fallbacks for different environments

/* style.css */

body {
  font-family: 'Roboto, -apple-system, BlinkMacSystemFont, "Segoe UI",
                Helvetica, Arial, sans-serif,
                "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol"',
}

Code: Self Hosted

Load Fonts from your server

/* style.css */

@font-face{
  font-family: 'Hack Reactor Sans';
  src: url('hackreactor-sans.eot');
  src: url('hackreactor-sans.eot?#iefix') format('embedded-opentype'),
       url('hackreactor-sans.woff') format('woff'),
       url('hackreactor-sans.ttf') format('truetype'),
       url('hackreactor-sans.svg#hackreactor-sans') format('svg');
}

html {
  font-family: "Hack Reactor Sans", "Bitstream Vera Serif", sans-serif;
}
